## KEY UX PRINCIPLES (MUST ENFORCE)

### 1. Never show worker lists
The platform assigns professionals.
Choice overload kills conversion.

### 2. Max 3 choices per screen
Too many options reduces bookings.

### 3. Always show price early
Users want price clarity before committing.

### 4. Trust signals everywhere
Egyptian users care about safety.

Always show:
- Verified professionals
- Platform guarantee
- Jobs completed

### 5. Large mobile buttons
Most usage will be one-handed on phones.

---

## TOTAL BOOKING TIME TARGET

**User flow:**
Home → Service → Address → Time → Payment → Confirm

**Total time:** 30–45 seconds

## ADVANCED UX FEATURE (HIGHLY RECOMMENDED)

Add **photo upload before booking**.

**Example:**
"Show plumber the problem. Upload photo."

**Benefits:**
- Better price estimates
- Faster diagnosis
- Fewer disputes

## THE NORTH STAR EXPERIENCE

The user experience should feel like this:

1. User opens app
2. Tap "Fix Plumbing"
3. Tap "Continue"
4. Enter address
5. Tap "Confirm"
6. Done. Technician arrives.

**This is the gold standard.**
