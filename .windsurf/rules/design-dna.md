# Design DNA

**Golden Rule:** If it feels like work, it's wrong.

## Principles
1. **One Thing Per Screen** - One input per screen. Never combine.
2. **Zero Decision Fatigue** - AI decides, user confirms. Max 3 options.
3. **Instant Gratification** - Every tap = progress. Fast animations. No blank screens.
4. **Talk, Don't Type** - AI chat primary. Forms = one field at a time.
5. **No Dead Ends** - Clear next action. Back works. Errors offer solutions.
6. **Visual Hierarchy** - One primary action. Large touch targets. Clear size contrast.
7. **Clean Typography** - Limited font weights. Large readable text.
8. **3-Second Rule** - User understands any screen in 3 seconds. No tutorials.

## Visual Hierarchy Rules
1. **Size Contrast** - Primary: 32-42px, Secondary: 18-24px, Tertiary: 13-16px
2. **Weight Contrast** - Headings: 700, Body: 400-500, Captions: 400
3. **Color Contrast** - Primary text: neutral[950], Secondary: neutral[600], Tertiary: neutral[400]
4. **Spacing Scale** - Use 8px base. Vertical rhythm: 16-24-32-48px
5. **One Primary CTA** - Largest button, strongest color, bottom of screen
6. **Card Hierarchy** - Border + shadow for elevation. Padding: 24-32px
7. **Icon Size** - Primary: 40-48px, Secondary: 24-32px, Tertiary: 16-20px
8. **Letter Spacing** - Large headings: -0.5 to -1px, Buttons: +0.5px

## Forbidden
❌ Multi-field forms ❌ Dropdowns ❌ Tiny text/buttons ❌ >3 choices ❌ Nested modals ❌ Scrolling for main action ❌ Spinners without context ❌ Onboarding tutorials

## Required
✅ Full-screen inputs ✅ Bottom CTA ✅ Progress indicators ✅ Haptic feedback ✅ Smooth transitions ✅ Skeleton states ✅ Pull-to-refresh ✅ Swipe gestures
